начнем

Определитесь с версией, которая вам нужна.

Если вас интересуют эвенты, игра TFC:Reforged или Original (1.16.5), то вам необходима java 8. Если это Original (1.17.1) и выше (Original-A), то вам необходима java 16. Можно установить обе версии, если собираетесь использовать все.

скачивание

Перейдите по нужной ссылке и выберите вашу операционную систему.

Java 8

Java 16

Приведем пример наиболее распространенного варианта - Windows 64x. Выбираем вариант JRE в формате установщика msi.

После загрузки установите как обычную программу.
